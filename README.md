# 人话

## 项目概述
VideoCropper 是基于 ![Python](https://img.shields.io/badge/-Python-3776AB?style=flat-square&logo=python&logoColor=white) 开发的视频 ROI 区域裁剪工具，采用 [![GitHub License](https://img.shields.io/github/license/KirisameMarisa-DAZE/ipsc-video-analysis)](LICENSE) 开源协议，可通过源代码 `cut.py` 实现视频指定区域的精确裁剪。本工具要求 `Python 3.6+` 环境支持，推荐通过源码直接运行以获得最佳兼容性。

## 环境配置

### 必要组件
- `Python 3.6` 或更高版本
- 必需依赖库：
  ```bash
  pip install PyQt5 opencv-python numpy
  ```

### 安装方式
1. 访问 [Python 官网](https://www.python.org/downloads/) 安装最新版本
2. 推荐使用 [Anaconda](https://www.anaconda.com/download) 集成环境
3. 配置系统环境变量 PATH

### 技术支持
如需环境配置帮助，请参考 [菜鸟教程](https://www.runoob.com/python3/)

## 使用指南

### 标准运行方式
```bash
python cut.py
```

### 应用程序包（实验性）
我们为以下平台提供预编译包：
- Windows (x64/x86)
- macOS (Intel/Apple Silicon)
- Linux (x64/x86)

#### 注意事项
1. 所有预编译包均未进行代码签名
2. 系统可能触发安全警告
3. 各平台运行要求：
   - **Windows**：需手动允许程序执行
   - **macOS**：需执行终端命令解除限制
   - **Linux**：未经完整测试

### macOS 特别说明

#### 文件结构
```
VideoCropper_macOS_*/           # 解压文件夹
  ├── VideoCropper_macOS_*.app  # 主程序
  └── VideoCropper_macOS_*/     # 冗余文件（建议删除）
      ├── VideoCropper_macOS_*  # Unix 可执行文件
      └── _internal/            # 依赖资源
```

#### 安全授权
```bash
# 解除系统限制
sudo xattr -cr /Applications/VideoCropper_macOS_*.app
# 设置执行权限
sudo chmod -R 755 /Applications/VideoCropper_macOS_*.app
```
